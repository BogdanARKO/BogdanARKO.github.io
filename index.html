CSS (каскадные таблицы стилей) в Википедии могут встречаться в трёх местах. В викиразметке разрешены только инлайн-стили (встроенные в строку), например span style="border:1px solid gray">текст/span>, с некоторыми дополнительными ограничениями, например, нельзя использовать background:url(...), эмбеддед-стили (встроенные отдельно на странице) в виде span {...} использовать нельзя. подробнее см. ВП:Викиразметка. В глобальном оформлении используются для изменения оформления значительного числа страниц, размещаются в виде внешних таблиц MediaWiki:Common.css и редактируются участниками с флагом администратора; подробнее см. ВП:Глобальный код. В персональном оформлении позволяют зарегистрированному участнику использовать для себя любые правила, размещаются в виде внешних таблиц на подстранице common.css участника только самим участником; подробнее см. ВП:Персональное оформление.
